# Sistema Académico UMB

## Descripción

Prototipo de una interfaz accesible para un sistema académico desarrollado en Figma aplicando principios de UX, UI y accesibilidad web.

## Tecnologías utilizadas

- Figma
- Microsoft Word
- GitHub

## Principios de accesibilidad

- Navegación mediante teclado.
- Breadcrumbs.
- Skip Link.
- Landmarks ARIA.
- Contraste adecuado.
- Componentes reutilizables.
